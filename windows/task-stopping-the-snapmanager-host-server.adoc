= Detener el servidor host SnapManager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Cuando termine de utilizar SnapManager, quizás desee detener el servidor.

. Para detener el servidor, escriba el siguiente comando, como usuario raíz:
+
`smsap_server stop`
